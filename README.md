# Kratos Base Project

本项目为一个使用 [kratos框架](https://github.com/go-kratos/kratos) 创建的，基础的微服务项目模板，
以便于后续快速开发。

## 项目目录

```
├── api // api proto文件和生成文件
├── app // 服务集合
│   ├── admin // 管理端接口服务
│   ├── auth // 权限服务
│   └── interface // 用户端接口服务
├── pkg // 自定义包
├── third_party // 第三方包
```
